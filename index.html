Autonomous security pipeline

Scan. Fix. Verify.

Not just reports — actual code fixes. Your agent scans a target, receives fix diffs for every vulnerability, applies them, and re-scans to verify. Fully autonomous, no human required.

01 SCAN

Agent calls quick_scan or full_audit

Via MCP protocol or REST API. Provide a target URL, get a structured vulnerability report with CVSS scores in minutes.

02 FIX

Get auto_fix with code diffs

17+ fix patterns across multiple frameworks. Receive actual unified diffs that your agent can apply directly. Not suggestions — real patches.

03 VERIFY

Apply fixes and re-scan

Agent applies the diffs, triggers another scan to confirm remediation. Zero vulnerabilities remaining — verified and done.

POST /api/scan Submit a scan request

Request body

{
  "target_url": "string",       // required
  "scan_type": "string",        // "quick" | "full"
  "callback_url": "string",     // webhook endpoint
  "api_key": "string"           // required for paid tiers
}

Quick start

terminal
# Free scan via CLI
npx @0xaudit/scanner https://your-app.com

# Or via API
curl -X POST https://0-x-audit.com/api/scan \
  -H "Content-Type: application/json" \
  -d '{"target_url": "https://your-app.com", "scan_type": "quick"}'

# Get auto-fix diffs (requires API key)
curl -X POST https://0-x-audit.com/api/auto-fix/scn_4f8a2b \
  -H "Authorization: Bearer your_api_key"

# Or connect via MCP (recommended for agents)
# Server: https://mcp.0-x-audit.com/sse
